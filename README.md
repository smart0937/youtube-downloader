# YouTube Downloader (Hermes Agent Skill)

This repository contains the backup of the `youtube-downloader` skill for the Hermes Agent.

## 📌 Overview
A specialized skill that allows the AI agent to download YouTube videos and subtitles based on a set of pragmatic engineering defaults.

## ✨ Key Features
- **Customizable Resolution**: Supports 720p, 1080p, or Best quality.
- **Flexible Paths**: Integrated with WSL to save files directly to Windows drives (e.g., `F:\Download\YouTube`).
- **Smart Subtitles**: 
  - Automatically fetches native or auto-generated subtitles.
  - Priority: Traditional Chinese, Simplified Chinese, English.
  - Converts subtitles to standard `.srt` format.
- **AI Translation**: Supports high-quality LLM-based translation for English subtitles.

## 🛠 Implementation
The skill leverages `yt-dlp` and `ffmpeg` to handle high-quality merging and subtitle conversion.
