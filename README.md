# YouTube Downloader (YouTube 下載器)

本技能用於將 YouTube 影片及字幕下載至本地硬碟。

## ⚙️ 預設設定
- **快捷操作**: 直接傳送網址 $\rightarrow$ 自動下載 **720p / MP4 / 無字幕**。
- **儲存路徑**: `F:\Download\YouTube` (WSL: `/mnt/f/Download/YouTube`)
- **命名規則**: `標題 [ID].mp4` (唯一命名法)

## 🚀 使用方式
1. **預設下載**: 直接提供影片網址即可。
2. **自定義需求**: 若有特殊要求（如：最高畫質、僅音訊、需要字幕），請在網址後一併告知。

## 🛠️ 技術備註 (針對 429 錯誤、空間管理與畫質問題)
如果在下載過程中遇到 `HTTP Error 429: Too Many Requests`，這通常是因為 YouTube 的字幕端點限制較嚴格。
**解決方案**：
1. 嘗試關閉字幕下載，僅下載影片本體。
2. 使用最精簡的格式選擇 (`-f "best"`)。
3. 等待短時間後重新嘗試。

**畫質問題 (解析度低於預期)**：
若下載結果解析度低於預期（例如 720p 變成了 360p），通常是因為 YouTube 的 JS 挑戰或直式影片（Shorts）的解析度定義導致。系統目前已配置 **Node.js Runtime** 並採用 **寬高雙向篩選** 邏輯，能確保橫式與直式影片均能穩定獲取 720p 以上的高畫質。

**空間管理 (片段截取)**：
當要求截取特定時間段時，系統會先將影片下載至 `/tmp/` 暫存，截取完成後會**立即自動刪除**暫存檔，以確保不會佔用您的磁碟空間。

## 📁 檔案路徑
- **WSL 路徑**: `/mnt/f/Download/YouTube`
- **Windows 路徑**: `F:\Download\YouTube`
